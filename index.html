Shortcuts

  • F: Next function preset
  • G: Previous function preset
  • T / Enter: Focus function input
  • Tab: Focus & lock canvas
  • Escape: Release focus/cursor
  • ZQSD/WASD: Move camera
  • Arrow Keys: Rotate shape
  • Ctrl + +/-: Adjust Nx/Ny
  • Ctrl + Arrow Keys: Adjust boundaries
